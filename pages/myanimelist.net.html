Myanimelist.net

Information de base

Myanimelist.net est un site de base de données sur les animations japonaises.

La couleur de fond est blanche

Les couleurs de textes sont:

  • Noir pour le texte
  • Bleu pour les liens
  • Blanc pour les textes et les liens derière un fond sombre

Le menu de navigation est au haut de la page et est un menu déroulant verticale. Ce menu nous permet d'accéder à la base de donnée d'anime et de mangas, le forum, l'industrie, les visionnements et le centre d'aide

La page d'acceuil mesure 3¼ écrans sur une résolutions de 1920x1080

La page d'acceuil contient trois publicitées au hasard de Google et une publisité de leur sponsor Crunchyroll. Une publicité est situé dans la colonne de droite et deux dans le bas de la colone de gauche.
