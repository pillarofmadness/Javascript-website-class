Anidb.net

Information de base

Anidb.net est un site de base de données sur les animations japonaises.

La couleur de fond est grise

Les couleurs de textes sont:

  • Noir pour le texte
  • Rouge et bleu pour les liens
  • Blanc pour les liens de navigation

Le menu de navigation est à gauche de la page et est un menu vertical. Ce menu nous permet d'accéder à la recherche, les listes d'anime, personnages, studios, créateurs etc.., La page de support, le forum, le clavardage et la page de connextion ou d'enregistrement

La page d'acceuil mesure 2½ écrans sur une résolutions de 1920x1080

La page d'acceuil contient aucune publicités.

Les images de prisentation d'anime s'agrandissent losqu'elles sont survolées
